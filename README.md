Customer Due Diligence (CDD) & Risk Assessment Dashboard

Overview
This project demonstrates a Customer Due Diligence (CDD) and Risk Assessment Dashboard developed using Power BI. It is based on a fictional banking dataset created for learning and portfolio purposes. The dashboard helps visualize customer information, KYC compliance, risk levels, PEP screening, sanctions screening, and customer investigation details to support AML compliance.

Project Objective
The objective of this project is to simulate a real-world Customer Due Diligence process by analyzing customer data and presenting meaningful insights through interactive dashboards. It focuses on identifying customer risk, monitoring KYC compliance, supporting PEP and sanctions screening, and assisting compliance teams in customer investigations.

Dataset
A fictional banking dataset containing 200 customer records was created for this project. The dataset includes customer demographics, KYC status, risk level, due diligence category, PEP status, sanctions screening, annual income, occupation, source of funds, and review status.
